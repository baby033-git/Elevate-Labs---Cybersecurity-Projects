# Password Strength Analyzer & Custom Wordlist Generator

This project analyzes password strength and generates a custom wordlist using user details.  
It checks password complexity, calculates entropy, and creates attack-style wordlists with
leetspeak variations. This demonstrates how weak passwords are exploited in dictionary attacks.
